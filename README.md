# 🎨 Applitex - Elite Frontend Interface

Applitex Frontend es una interfaz de usuario de "Nueva Generación" diseñada para ofrecer una experiencia fluida, estética y altamente productiva en la gestión industrial. Basada en una estética **Elite Modern**, la aplicación combina potencia de datos con una visualización minimalista y reactiva.

---

## 🌟 Experiencia de Usuario (UX)

Applitex no solo funciona bien; se siente increíble.
- **Glassmorphism**: Uso sutil de transparencias y desenfoques para una profundidad visual superior.
*   **Micro-interacciones**: Transiciones suaves y feedbacks visuales en tiempo real para cada acción.
*   **Responsive Pro**: Adaptabilidad total desde monitores de alta resolución hasta dispositivos móviles para el personal de planta.

---

## 🛠️ Stack Tecnológico

- **Framework**: React 18 con Vite para un desarrollo ultra-rápido.
- **Lenguaje**: TypeScript para un tipado estricto y seguro.
- **Estilos**: Tailwind CSS 4 con una arquitectura de tokens personalizados.
- **Estado**: Zustand para una gestión de estado global ligera y escalable.
- **Iconografía**: Lucide Icons para una estética técnica y limpia.
- **Navegación**: React Router 6 con guardias de autenticación.
- **Comunicación**: Axios con interceptores para gestión automática de Tokens JWT y Multi-tenancy.

---

## 🏗️ Módulos de la Interfaz

### 📊 Dashboard Holístico
Visualización rápida del estado de la empresa con indicadores clave (KPIs) y acceso directo a las tareas críticas.

### 👥 Centro de Clientes
Gestión de perfiles de clientes con segmentación técnica y comercial.

### 📂 Catálogo Técnico
Explorador de productos con gestión de fichas técnicas dinámicas. Una interfaz potente para configurar la ingeniería de cada producto.

### 📦 Inventario Inteligente
Visualización de existencias con filtros avanzados y autogeneración de referencias. Diseñado para simplificar la toma de inventarios.

### 📝 Órdenes de Servicio (Core)
El punto de convergencia. Una interfaz interactiva que integra catálogo e inventario para gestionar la producción con precisión milimétrica.

## 🚀 Instalación y Desarrollo

### Requisitos Previos
- Node.js 18+ instalado.
- npm o yarn.

### Configuración del Entorno
Crea un archivo `.env` en la raíz del proyecto:

```env
VITE_API_URL=http://localhost:8080/api/v1
```

### Comandos Útiles
```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Construir para producción
npm run build
```
